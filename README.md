## umair.codes


Code for my personal website and blog built with React

#### What's inside?

[create-react-app](https://github.com/facebook/create-react-app) as the base with the following dependencies:

- react
- react-dom
- react-helmet
- react-router-dom
- markdown-with-front-matter-loader
- github-markdown-css

#### Commands

- `npm start`
- `npm test`
- `npm run build`
- `npm run eject`
