## umair.codes


Code for my personal website built with React

#### What's inside?

[create-react-app](https://github.com/facebook/create-react-app) as the base with the following dependencies:

- react-helmet
- react-router-dom
- react-snap
- react-ga


#### Commands

- `npm start`
- `npm test`
- `npm run build`
- `npm run eject`
